// ⭐ IDIOMS & PHRASES (محاورے اور جملے کے اظہار)  
// English + Urdu hints

Vocabulary
│
├─ 1️⃣ Definition  
│       define: **Idioms** are fixed expressions with meanings different from the literal words.  
│       **Phrases** are groups of words that convey a particular meaning.  
│       Urdu: **Idioms** → ایسے جملے جو لفظی معنی سے مختلف معنی رکھتے ہیں  
│             **Phrases** → الفاظ کا گروہ جو ایک خاص مطلب دیتا ہے  

├─ 2️⃣ Examples  
│
│   1. Break the ice → **To initiate conversation in a friendly way**  
│      // ماحول دوستانہ بنانے کے لیے بات چیت شروع کرنا  
│
│   2. A piece of cake → **Very easy**  
│      // بہت آسان  
│
│   3. Hit the nail on the head → **To describe exactly what is causing a situation**  
│      // بالکل صحیح بات کہنا یا اصل مسئلے کی نشاندہی کرنا  
│
│   4. Under the weather → **Feeling sick or unwell**  
│      // طبیعت خراب ہونا  
│
│   5. Kill two birds with one stone → **Achieve two goals with one action**  
│      // ایک ہی کام سے دو فائدے حاصل کرنا  

// ✅ Summary (خلاصہ)  
// 1. Idioms → **fixed expressions with figurative meanings**  
// 2. Phrases → **group of words conveying a meaning**  
// 3. Idioms & Phrases → improve **fluency, writing, and spoken English**  
// 4. Urdu tip: محاورے → idioms, جملے کے اظہار → phrases  
// 5. Usage tip: Use idioms & phrases to **make language more natural and expressive**
