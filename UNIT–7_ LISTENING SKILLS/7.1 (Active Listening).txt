// ⭐ ACTIVE LISTENING (فعال سماعت)

Active Listening
│
├─ 1️⃣ Definition  
│       define: **The skill of fully concentrating, understanding, and responding to a speaker**  
│       Urdu: **بولنے والے کی بات کو پوری توجہ سے سننا، سمجھنا اور مناسب ردِعمل دینا**  

├─ 2️⃣ Purpose of Active Listening  
│       • To understand the message clearly  
│         // پیغام کو واضح طور پر سمجھنا  
│       • To build trust and respect  
│         // اعتماد اور احترام قائم کرنا  
│       • To avoid misunderstandings  
│         // غلط فہمیوں سے بچنا  

├─ 3️⃣ Key Elements of Active Listening  
│
│   1. Attention (توجہ)  
│       define: **Focusing completely on the speaker**  
│       Urdu: **بولنے والے پر مکمل توجہ دینا**  
│
│   2. Understanding (سمجھ بوجھ)  
│       define: **Interpreting the message correctly**  
│       Urdu: **پیغام کو درست طور پر سمجھنا**  
│
│   3. Response (ردِعمل)  
│       define: **Giving feedback through words or gestures**  
│       Urdu: **الفاظ یا اشاروں سے جواب دینا**  
│
│   4. Feedback (فیڈبیک)  
│       define: **Confirming understanding of the message**  
│       Urdu: **پیغام کی سمجھ کی تصدیق کرنا**  

├─ 4️⃣ Techniques of Active Listening  
│       • Maintain eye contact  
│         // آنکھوں کا رابطہ برقرار رکھیں  
│       • Nod or use facial expressions  
│         // سر ہلانا یا چہرے کے تاثرات  
│       • Do not interrupt  
│         // بات کے درمیان نہ بولیں  
│       • Ask relevant questions  
│         // مناسب سوالات پوچھیں  

├─ 5️⃣ Benefits  
│       • Improves communication  
│         // گفتگو کو بہتر بناتا ہے  
│       • Strengthens relationships  
│         // تعلقات مضبوط کرتا ہے  
│       • Enhances problem-solving  
│         // مسئلہ حل کرنے کی صلاحیت بڑھاتا ہے  

└─ ✅ Summary (خلاصہ)  
       1. Active Listening → **Listening with full attention and understanding**  
       2. Purpose → Clear understanding and trust  
       3. Techniques → Eye contact, feedback, no interruption  
       4. Urdu tip: فعال سماعت → غور سے سننے کی مہارت  
       5. Tip: **Listening is as important as speaking**
